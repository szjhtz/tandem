---
title: Desktop First 10 Minutes
---

Use this checklist to go from install to a working prompt flow quickly.

## 1. Launch and Connect

1. Open Tandem Desktop.
2. Confirm engine status shows connected.
3. If prompted, complete provider setup.

## 2. Configure a Provider

1. Open provider settings.
2. Add API key for your chosen provider.
3. Set a default provider and model.

## 3. Open Your Workspace

1. Open a project folder.
2. Start a new session.
3. Send a first prompt such as:

```text
Summarize this repository and list the top 3 areas to improve.
```

## 4. Handle Requests Safely

1. Review pending tool requests.
2. Approve only what is needed.
3. Use deny/ask-again when a request is unclear.

## 5. Validate End-to-End

1. Confirm assistant response includes repo-specific details.
2. Run one command/tool-assisted action.
3. Save session title for future recall.
